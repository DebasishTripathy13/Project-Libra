Integrates Gemini, OpenAI, and Ollama to not just read logs, but hunt for threats, explain anomalies, and suggest remediation.

ML Anomaly Detection

Built-in IsolationForest and Baseline Learning models adapt to your system's normal behavior to spot zero-day anomalies.

Metasploit-Style Console

Powerful interactive CLI for security professionals. Run modules, manage database, and hunt threats from the terminal.
